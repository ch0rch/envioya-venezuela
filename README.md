# EnvíoYA Venezuela

Herramienta web para ayudar a quienes están en el exterior a hacer llegar dinero a sus familias en Venezuela de la forma más conveniente, vía **Pago Móvil**.

Una iniciativa personal en respuesta a los terremotos de junio de 2026 en Venezuela.

## Qué hace

- Elegís **desde qué país o moneda enviás** y **cuánto**.
- Te muestra **la mejor forma de que les lleguen más bolívares** vía Pago Móvil: cuánto llega (estimado), cuánto tarda, y qué necesita quien recibe — solo **teléfono + cédula**, sin ir al banco.
- Podés **comparar y elegir el método que ya usás** (cripto, billetera o transferencia).
- Un botón te lleva a **Saldoar** con el envío pre-cargado para completarlo.
- Incluye una sección con **canales de donación verificados** para ayudar a las personas afectadas por el terremoto.

## Por qué

En una emergencia, la gente necesita mandar plata rápido y que llegue lo máximo posible. Esta herramienta muestra, de forma clara y honesta, cuál es la mejor opción para cada caso. Los montos son **estimados**; el monto final se confirma al completar el envío.

## Tecnología

- [Astro](https://astro.build) (renderizado en el servidor) + TypeScript
- Desplegado en [Vercel](https://vercel.com)

## Desarrollo

```bash
npm install
npm run dev      # http://localhost:4321
npm test
```

## Sobre el proyecto

Proyecto personal sin fines de lucro, hecho de forma independiente para ayudar en la emergencia. La cotización y el envío del dinero los provee y ejecuta **[Saldoar](https://saldoar.com)**; esta herramienta solo ayuda a elegir la mejor opción y deriva a Saldoar para concretar la operación.
